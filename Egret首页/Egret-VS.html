Egret Micro Client

Egret 微端是一种 H5 游戏的原生打包方案 初始包体小、易推广、存留远超传统具备多网络启动模式和资源缓存功能易用的第三方 SDK 接入方案

使用产品 教程文档 产品白皮书

  • Runtime 下载器

    根据 android 设备芯片架构下载对应的最小的 Runtime

  • WebView

    支持各种引擎制作的 Html5 游戏

  • Runtime

    完善的资源管理机制，纯原生渲染，可以让白鹭制作的游戏实现绝佳的运行效果，流畅度超越浏览器

极小包体不到2M

传统打包方案

  • 包体太慢，下载速度慢
  • 玩家下载转换率低
  • 渠道推广意愿不强

Micro Client打包方案

  • 初始包体小，2秒极速下载
  • 包体不到2M，极易渠道推广
  • 占用空间小，更易存留在用户手机

游戏配置功能

游戏图标

供系统使用不同尺寸的图标

加载背景

设置下载 Runtime 时显示的背景图，游戏体验更好

游戏链接

后台动态设置游戏的链接地址，方便游戏调整

第三方 SDK

可方便接入广告、支付宝等第三方 SDK
